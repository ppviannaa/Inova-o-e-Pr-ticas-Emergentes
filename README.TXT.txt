Este projeto consiste em uma calculadora básica criada em Python que visa fornecer uma aplicação útil para o aprendizado de programação. A calculadora tem uma interface gráfica agradável e pode realizar operações matemáticas básicas.

Necessidades:Certifique-se de ter o Python instalado em seu sistema. Você pode baixá-lo em python.org.

Clone o repositório para sua máquina local: https://github.com/ppviannaa/Inovacao-e-Praticas-Emergentes.git

Navegue até o diretório do projeto:cd Inovacao-e-Praticas-Emergente; cd Calculadora.py

Execute o script Python:python Calculadora.py
